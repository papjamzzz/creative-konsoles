# Creative Konsoles — Website

**creativekonsoles.com** · Hosted on GitHub Pages

## Deploy in 3 steps

1. `git init && git add . && git commit -m "initial"`
2. Push to a new GitHub repo named `creative-konsoles`
3. Settings → Pages → Source: `main` branch → Save

Add your custom domain in Settings → Pages → Custom domain: `creativekonsoles.com`
Then add a CNAME record in Namecheap pointing to `papjamzzz.github.io`

## Swap the music
Replace `assets/perfect-to-me.mp3` with your actual track.
