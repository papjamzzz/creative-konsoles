# Mac Automation Suite — Jeremiah Stephen Smith
## Built for: 10hr/day Claude + Audio Production workflow on M1 MacBook Pro

---

## What's In Here

| File | What It Does |
|------|-------------|
| `focus_mode.sh` | Kills distractions, sets Do Not Disturb, launches your work apps |
| `audio_mode.sh` | Switches to audio production mode — launches Ableton, sets buffer |
| `quick_note.sh` | One command to capture an idea to Apple Notes instantly |
| `project_snapshot.sh` | Saves a daily snapshot of what you're working on |
| `cleanup_downloads.sh` | Auto-sorts your Downloads folder |
| `automations.applescript` | Keyboard shortcut automations for common tasks |
| `install.sh` | Run this once to set everything up |

---

## Installation

Open Terminal and run:
```bash
bash ~/Desktop/mac-automations/install.sh
```

That's it. All commands will be available immediately.

---

## Daily Commands

```bash
focus          → Kill distractions, launch Claude + Chrome
audio          → Launch Ableton, optimize for audio
idea "text"    → Save an idea to Apple Notes instantly
snapshot       → Save today's work status
cleanup        → Sort Downloads folder
```
